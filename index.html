Technical Arsitektur Projek Utama

Arsitektur & Alur Solusi Sistem IoT FlowBeat

1. Stack Teknologi

Sistem dibangun penuh menggunakan teknologi modern terintegrasi:

React Native Express.js (Node.js) AWS Cloud Ecosystem MySQL Backend

2. Alur Transmisi Data Pintar

Bagaimana data kesehatan pasien mengalir di dalam sistem:

  1. Sensor IoT Hardware: Membaca detak jantung (BPM) dan SpO2 dari tubuh pasien lansia.
  2. API Gateway & Lambda Gateway: Mengamankan serta meneruskan beban data (*payload*) enkripsi ke server database.
  3. ExpressJS Router: Mengelola logika pengolahan data *time-series* dan validasi autentikasi token.
  4. React Native UI: Melakukan visualisasi grafik kesehatan interaktif untuk dianalisis oleh tenaga kesehatan secara presisi.

Catatan Pengembang

Website portofolio ini menerapkan arsitektur *clean code*. Semua komponen antarmuka dirender secara dinamis memanfaatkan objek JSON statis untuk menjamin kecepatan pemuatan situs yang optimal di perangkat seluler.
